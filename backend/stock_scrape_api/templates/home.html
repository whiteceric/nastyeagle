Welcome to the nastyeagle API

How to Use:

To find the current price of a stock, make a GET request to nastyeagle.com/stock-price/[STOCK_TICKER]. For example, the current price of $TSLA can be found with:

curl www.nastyeagle.com/stock-price/TSLA

For which the response will be a JSON like:


  {
    "ticker": "TSLA",
    "name": "Tesla Inc. Common Stock",
    "latest_price": 609.89,
    "latest_day_change": -0.23,
    "last_updated": 1623545370.135269
  }

Note that 'last_updated' represents the timestamp for which the stock was last updated (updates do not happen at each API call, only when necessary for an up to date price)
